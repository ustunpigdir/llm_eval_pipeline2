**1. Compute the Lorentz factor $\gamma$ at $v=0.8c$.**

The Lorentz factor $\gamma$ is given by:
\[
\gamma = \frac{1}{\sqrt{1 - v^2/c^2}}
\]
Plugging in $v = 0.8c$:
\[
\gamma = \frac{1}{\sqrt{1 - (0.8)^2}} = \frac{1}{\sqrt{1 - 0.64}} = \frac{1}{\sqrt{0.36}} \approx 1.291
\]

In the rockets' final rest frame, the proper separation is the same as the rest length of the string, which is $100$ m. This is because in their rest frame, they are not moving relative to each other.
