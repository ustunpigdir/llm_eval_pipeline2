The Hubble distance is obtained from \(c/H_{0}\).  
Using \(c = 2.99792\times10^{5}\ {\rm km\,s^{-1}}\) and \(H_{0}=70.0\ {\rm km\,s^{-1}\,Mpc^{-1}}\),

\[
\frac{c}{H_{0}}=\frac{2.99792\times10^{5}}{70.0}\ {\rm Mpc}
               = 4282.742857\ {\rm Mpc}\;\approx\;4282.74\ {\rm Mpc}.
\]

The linear term at \(z=0.200\) is

\[
\frac{c z}{H_{0}} = \frac{c}{H_{0}}\;z
                  = 4282.742857 \times 0.200
                  = 856.548571\ {\rm Mpc}\;\approx\;856.55\ {\rm Mpc}.
\]

For the LCDM model with \(q_{0}=-0.55\),

\[
1-q_{0}=1-(-0.55)=1.55,\qquad
\frac{(1-q_{0})z}{2}= \frac{1.55\times0.200}{2}=0.155,
\]
\[
\text{bracket}=1+0.155=1.155,
\]
\[
d_{L}^{\rm LCDM}= \frac{c z}{H_{0}}\times1.155
                = 856.548571\times1.155
                = 989.313600\ {\rm Mpc}\;\approx\;989.31\ {\rm Mpc}.
\]

For Einstein–de Sitter with \(q_{0}=+0.5\),

\[
1-q_{0}=1-0.5=0.5,\qquad
\frac{(1-q_{0})z}{2}= \frac{0.5\times0.200}{2}=0.05,
\]
\[
\text{bracket}=1+0.05=1.05,
\]
\[
d_{L}^{\rm EdS}= \frac{c z}{H_{0}}\times1.05
                = 856.548571\times1.05
                = 899.376000\ {\rm Mpc}\;\approx\;899.38\ {\rm Mpc}.
\]

A supernova at \(z=0.2\) that appears fainter than the EdS prediction would have a larger distance modulus, i.e. a larger \(d_{L}\) than the EdS value of \(\sim 899\) Mpc. This implies a smaller deceleration parameter (more negative \(q_{0}\)), consistent with the LCDM value \(q_{0}=-0.55\). The observed faintness therefore favours an accelerating universe, which was the key evidence for the 1998 discovery of cosmic acceleration and the need for dark energy.
